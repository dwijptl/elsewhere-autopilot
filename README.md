# Bharat Ke Rahasya — elsewhere-autopilot

Fully automated faceless YouTube channel: **Indian & dharmic mystery
investigation documentaries in Hindi** (working identity: भारत के रहस्य —
rename in `config.yaml` → `brand.name`).

Same engine as `faceless-autopilot` (channel 1), converted to the mystery
genre: five-act investigation scripts (दावा → स्रोत → इतिहास → सिद्धांत →
जो बचा है), a documented-timeline story meter, three labeled registers
(शास्त्र / इतिहास / विज्ञान), respect-first dharmic guardrails, honest
unresolved endings, topic-driven visual identity (30 style packs — mystery
topics land in folklore/manuscript/relic/noir/archive), grounded
fact-checking, Sarvam Hindi voice.

Read `docs/MYSTERY_GENRE_STUDY.md` for WHY this genre works and the rules
this pipeline encodes. Read `docs/PROJECT_LOG.md` before changing anything.

Workflows: `Make Video`, `Make Short` (manual `topic` input supported),
`CI`, `Test Voice`, `Update Learnings`. Secrets required: GEMINI_API_KEY,
PEXELS_API_KEY, SARVAM_API_KEY (optional: FAL_KEY, ANTHROPIC_API_KEY).
